Image Captioning

This task is significantly harder in constrast to well-researched tasks such as image classification and object recognition. A solution to this task would provide end-user utility when developing ADAS systems. The challenge lies in the image captioning model having to overcome both computer vision obstacles as well as extracting relationships in natural language [17] .

Thus, the problem of image captioning can be summarized as

  • visual feature extraction for encoding
  • decoding of features to natural language

An approach towards solving the task follows an Encoder-Decoder Framework [16], [18] utilizing 2 components

  • Convolutional Neural Networks (CNN) - encode images into visual features
  • Recurrent Neural Network (RNN) [14], [15], [16] - decode visual features to descriptions
